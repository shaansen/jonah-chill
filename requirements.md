# Requirements for this app.

1. PWA - Progressive Web App that I'll "Add to Home Screen" on my iPhone Safari to create an app.
2. This will be a Github repo and I'll enable it on Github Pages using Actions.
3. This app will let me upload an epub file and will convert it into an audiobook for me to listen to.
4. This will be local to my device that means it will only read the epub file from my device and will not access the internet.
5. It will also save my progress on how much I've listened to the audiobook locally on my device.
6. Nice to have configuration to control the speed or voice of the audiobook.
7. Nice to have a way to move to different chapters in the audiobook.
8. make the SEO for this repo such that its as close to unsearchable
